## 二、GEO分析工具 MVP

### 核心功能
用户粘贴文章 → 分析GEO得分 → 给出优化建议

### 评分维度（基于9大GEO原则）
1. **引用密度** - 是否有权威来源引用
2. **数据支撑** - 是否包含统计数据
3. **专家引述** - 是否有专家观点
4. **可读性** - 段落长度、句子复杂度
5. **直接回答** - 是否有明确的答案/结论
6. **结构清晰** - 标题层级、列表使用
7. **权威信号** - 作者信息、来源可信度
8. **内容新鲜度** - 是否有时效性内容
9. **Schema标记** - 是否有结构化数据建议

### 技术方案
```
前端：文本输入框 + 实时分析展示
后端：API分析接口
分析：可用AI辅助（调用LLM分析内容质量）
```

### 页面设计
- `/tools/analyzer` - GEO内容分析器
- 输入区：粘贴文章内容
- 输出区：
  - 总分（0-100）
  - 各维度得分雷达图
  - 具体优化建议列表
  - 优化后预览（可选）
