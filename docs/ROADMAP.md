# DeepZD 产品路线图

> 从 GEO 知识门户到 GEO SaaS 工具

## 当前状态

- ✅ 知识门户网站
- ✅ GEO 概念教育内容
- ✅ 博客系统
- ✅ 多语言支持（中/英）

## 目标愿景

打造国内领先的 GEO（生成式引擎优化）工具平台，帮助内容创作者和企业提升在 AI 搜索引擎中的可见度。

---

## Phase 1：内容分析器（MVP）

**目标**：让用户体验到价值，验证市场需求

**预计周期**：2-4 周

### 核心功能

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 内容输入 | 支持粘贴文本 / 输入 URL | P0 |
| AI 友好度评分 | 0-100 分，多维度评估 | P0 |
| 优化建议 | 结构、引用、关键词等具体建议 | P0 |
| 一键复制 | 复制优化后的内容 | P1 |

### 评分维度

- 结构清晰度（标题层级、段落划分）
- 引用权威性（是否有数据、来源）
- 答案直接性（是否直接回答问题）
- 实体丰富度（专有名词、品牌、人名）
- 内容完整性（覆盖相关子话题）

### 技术方案

- 前端：现有 Next.js 框架
- 后端：Supabase Edge Functions
- AI：调用 Claude/GPT API 进行分析

---

## Phase 2：问题库 & 热门话题

**目标**：积累数据资产，为用户提供内容方向

**预计周期**：4-6 周

### 核心功能

| 功能 | 说明 | 优先级 |
|------|------|--------|
| AI 热门问题 | 用户在问 AI 什么问题 | P0 |
| 行业分类 | 按领域/行业分类问题 | P0 |
| 趋势追踪 | 问题热度变化趋势 | P1 |
| 内容建议 | 针对热门问题的写作建议 | P1 |

### 数据来源

- 公开 AI 对话数据
- 用户贡献
- 爬虫采集（合规）

---

## Phase 3：品牌监控（核心付费功能）

**目标**：监测品牌在 AI 中的表现，核心商业化功能

**预计周期**：6-8 周

### 核心功能

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 品牌提及监控 | 品牌被 AI 提到多少次 | P0 |
| AI 引用来源 | 哪些内容/网站被引用 | P0 |
| 引用排名 | 在同类查询中排第几 | P0 |
| 提及上下文 | AI 在什么语境下提到品牌 | P1 |
| 情感分析 | 正面/负面/中性提及 | P2 |

### 技术挑战

- 大规模 AI 对话数据采集
- 实时监控能力
- 数据存储和查询性能

---

## Phase 4：竞争分析 & 企业功能

**目标**：企业级功能，提高客单价

**预计周期**：8-12 周

### 核心功能

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 竞争对手对比 | 和竞品的 AI 曝光对比 | P0 |
| 多平台分析 | ChatGPT/Claude/Perplexity 分别表现 | P0 |
| 趋势报告 | 自动生成周报/月报 | P1 |
| API 接口 | 供企业集成使用 | P1 |
| 团队协作 | 多账号、权限管理 | P2 |

---

## 商业模式

### 定价策略（草案）

| 版本 | 价格 | 功能 |
|------|------|------|
| 免费版 | ¥0 | 内容分析（有限次数）、问题库浏览 |
| Pro 版 | ¥99/月 | 无限分析、品牌监控（1个品牌） |
| 企业版 | ¥499/月 | 多品牌、竞争分析、API、团队协作 |
